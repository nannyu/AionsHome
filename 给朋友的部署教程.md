## 🌟 功能概览

Aion Chat 是一个**局域网多端同步 AI 聊天系统**，支持 PC/手机浏览器同时使用，数据全部存在本地。

核心功能：
- **AI 聊天** — 支持多个 AI 模型（Gemini / 硅基流动 / 中转站），流式输出，多端实时同步
- **向量记忆库（RAG）** — AI 会自动总结和记忆你们的对话，下次聊天时能回忆起来
- **世界书** — 自定义 AI 和用户的人设、性格、说话方式
- **语音合成（TTS）** — AI 回复自动朗读，支持自定义音色
- **摄像头智能监控** — 定时截图 + AI 分析（哨兵/Core 双脑架构）
- **高德地图定位** — 手机 GPS 上报，AI 知道你在哪里（需要高德 Key）
- **日程/闹铃系统** — AI 可以帮你设置日程和闹铃，到时间自动提醒
- **音乐点歌** — AI 可以搜索网易云音乐并在线播放
- **设备活动日志** — PC 前台窗口 / 手机 App 使用情况记录，AI 可以查看你的动态
- **心语日记** — AI 的秘密日记，你可以偷看
- **PWA 支持** — 手机浏览器可以安装为独立 App（全屏无地址栏）
- **Android App** — 原生 WebView 壳，支持推送通知和语音录音（需自行编译）

## 🚀 第一步：安装 Python

Aion Chat 的后端用 Python 写的，所以你电脑上需要装 Python。

1. 打开 https://www.python.org/downloads/
2. 点那个大大的黄色按钮「Download Python 3.x.x」下载安装包
3. **运行安装程序时，最重要的两步：**

   ⚠️ **一定要勾选底部的 「Add Python to PATH」** ⚠️

   ![勾选示意](https://docs.python.org/3/_images/win_installer.png)

4. 然后点 **「Customize installation」**（自定义安装）
5. 在「Optional Features」页面，确认以下选项**都打勾了**：
   - ✅ pip
   - ✅ **py launcher**
   - ✅ **tcl/tk and IDLE**（可选）
6. 点 Next，在「Advanced Options」页面，确认以下选项**打勾**：
   - ✅ **Install for all users**（推荐）
   - ✅ **Add Python to environment variables**
   - ✅ **Precompile standard library**
7. 点 Install 等它装完

> ⚠️ **不要从 Microsoft Store 安装 Python！** Microsoft Store 版本的 Python 创建虚拟环境时经常出问题。请从 python.org 官网下载安装。

### 怎么确认装好了？

按 `Win + R`，输入 `cmd` 回车，在黑窗口里输入：
```
python --version
```
如果显示类似 `Python 3.12.x` 就说明装好了。如果显示「不是内部命令」，说明安装时没勾 PATH，需要重新安装。

## 🔩 第一步半：安装 C++ 编译工具（大概率需要）

有些 Python 库（如 `pywin32`）在安装时需要 C++ 编译环境。**如果你电脑上从来没装过 Visual Studio 或 C++ 开发工具，这一步必须做。**

1. 打开下载地址：https://visualstudio.microsoft.com/zh-hans/visual-cpp-build-tools/
2. 点 **「下载生成工具」** 下载 `vs_BuildTools.exe`
3. 运行安装程序，等它加载完后会出现一个选择界面
4. 勾选 **「使用 C++ 的桌面开发」**（Desktop development with C++）
5. 右侧可选组件保持默认即可，点 **「安装」**
6. 安装完成后**重启电脑**

> 💡 这个安装包比较大（约 2-5 GB），但只需要装一次。
> 如果你电脑上已经装了 Visual Studio（任何版本），并且安装时选了 C++ 相关组件，这一步可以跳过。

### 怎么确认装好了？

双击「一键安装环境.bat」，如果不再出现类似 `error: Microsoft Visual C++ 14.0 or greater is required` 的红色报错，就说明 OK 了。

## 🔧 第二步：安装依赖（只需要做一次）

1. 双击 **「一键安装环境.bat」**（这里如果出现乱码情况说明编码不兼容，请使用同目录文件夹下的「install.bat」）
2. 它会自动：
   - 检查你的 Python 是否装好
   - 检查虚拟环境模块是否可用
   - 创建虚拟环境（不会污染你电脑上其他 Python 程序）
   - 下载安装所有需要的库
3. 看到 **「✅ 环境安装完成！」** 就成功了
4. 如果中途报网络错误，检查一下网络连接，然后重新双击再跑一次就行

> 💡 这一步只需要做一次。以后启动不用再装了。

## ▶️ 第三步：启动 Aion Chat

### 手机上也想用？

电脑和手机连同一个 WiFi，然后：

1. 在电脑上按 `Win + R`，输入 `cmd`，然后输入 `ipconfig`

## 🔑 第五步：配置 API Key（必须！）

Aion Chat 需要 AI 模型的 API Key 才能和你聊天。

1. 打开聊天页面（`http://localhost:8080/chat`）
2. 点击右上角的 **⚙️ 齿轮图标**
3. 在设置面板里找到 **API Key** 区域
4. 填入你的 Key：

### 必填：Gemini Key（免费）

Gemini Key 是**必须**的，因为记忆库的向量数据库和哨兵模型都依赖它。

1. 打开 https://aistudio.google.com/apikey
2. 用 Google 账号登录（需要科学上网）
3. 点 「Create API Key」创建一个
4. 复制 Key，粘贴到设置面板的「Gemini Key」和「Gemini Free Key（哨兵+向量）」两个框里

> 💡 如果你有多个 Gemini Key，可以分开填：一个主聊天用，一个给哨兵+向量用（免费额度分开算）。只有一个的话两个框填一样的即可。

### 可选：硅基流动 Key

硅基流动提供国内模型（GLM / Kimi 等）和语音合成 TTS 功能。

1. 打开 https://cloud.siliconflow.cn
2. 注册登录后在控制台获取 API Key
3. 粘贴到设置面板的「硅基流动 Key」框里

> 不填也能用，只是不能使用硅基流动的模型和 TTS 语音播报。

### 可选：中转站 Key

支持 Claude 等模型的 AiPro 中转站（需付费）。

## ❓ 常见问题

### Q：双击 bat 文件闪退了怎么办？
右键 bat 文件 → 「编辑」或用记事本打开，在最后一行加上 `pause`，然后再双击运行，就能看到错误信息了。

### Q：安装依赖时报错 `pip 不是内部命令`？
Python 安装时没勾选 PATH。卸载 Python 重新安装，记得勾选 「Add Python to PATH」。

### Q：启动后浏览器打不开 localhost:8080？
- 确认黑色窗口还开着，没有报错
- 试试用 `http://127.0.0.1:8080` 访问
- 检查是否有其他程序占用了 8080 端口

### Q：手机访问不了电脑的 IP？
- 确认电脑和手机在同一个 WiFi 下
- 检查电脑防火墙是否放行了 8080 端口
- Windows 防火墙设置：控制面板 → Windows Defender 防火墙 → 高级设置 → 入站规则 → 新建规则 → 端口 → TCP 8080 → 允许连接

### Q：摄像头功能怎么用？
在聊天页的齿轮设置里开启。如果你不需要摄像头监控，不开就行，不影响聊天。

### Q：网络很慢，安装依赖失败？
可以用国内镜像加速。打开命令行（cmd），cd 到 AionsHome 文件夹，手动运行：
```
.venv\Scripts\pip install -r aion-chat\requirements.txt -i https://mirrors.aliyun.com/pypi/simple/
```

### Q：安装依赖时报错 `Microsoft Visual C++ 14.0 or greater is required`？
你的电脑缺少 C++ 编译工具。请按照上面「第一步半」的说明安装 **Microsoft C++ Build Tools**：
https://visualstudio.microsoft.com/zh-hans/visual-cpp-build-tools/

安装时勾选「使用 C++ 的桌面开发」，装完重启电脑，然后重新双击「一键安装环境.bat」。

### Q：创建虚拟环境失败 / 报错 `No module named venv`？
这通常是因为你的 Python 是从 **Microsoft Store** 安装的，或者安装时没有包含完整组件。解决方法：
1. **卸载** Microsoft Store 版本的 Python（设置 → 应用 → 搜索 Python → 卸载）
2. 从官网 https://www.python.org/downloads/ **重新下载安装**
3. 安装时选「Customize installation」，确保所有组件都勾选了
4. 重新双击「一键安装环境.bat」

### Q：pywin32 安装失败？
`pywin32` 是 Windows 专属库，用于获取电脑前台窗口信息（设备活动日志功能）。如果安装失败，不影响核心聊天功能，只是看不到 PC 端的活动日志。可以尝试：
```
.venv\Scripts\pip install pywin32
```

### Q：Gemini 连不上？
Gemini 是 Google 的服务，国内可能需要科学上网。确保你的网络能访问 `generativelanguage.googleapis.com`。

## � 编译 Android App（可选）

> AionApp 是一个 Android 原生 WebView 壳，比浏览器多了推送通知、语音录音权限、GPS 定位上报等功能。
> 如果你只用浏览器访问，这一节可以跳过。

### 1. 安装 Android Studio

1. 下载地址：https://developer.android.com/studio
2. 安装时保持默认选项即可，不需要修改
3. 首次启动会下载 Android SDK，需要**科学上网**或配置国内镜像（`dl.google.com` 在国内可能无法访问）
4. 等待「Android SDK」下载完毕，可能需要 10-30 分钟

> ⚠️ **Android Studio 自带 JDK 17**，不需要单独安装 Java。但如果你电脑上之前装过旧版 JDK（如 JDK 8/11），可能会冲突。
> 解决方法：打开 Android Studio → `File → Settings → Build → Gradle` → 确认 Gradle JDK 选的是 **Bundled（内置的 JDK 17）**。

### 2. 打开项目

1. 打开 Android Studio → 选 `Open` → 选择 `AionsHome/AionApp/` 文件夹
2. 等待 Gradle 同步完成（右下角进度条跑完，首次可能要下载依赖，需要几分钟）
3. 如果 Gradle 同步报错，看下面的「常见构建报错」

### 3. 修改服务器 IP 地址

App 里硬编码了服务器地址，你需要改成你自己的 IP。一共 **3 个文件**，都在 `AionApp/app/src/main/java/com/aion/chat/` 目录下：

#### ① `LauncherActivity.java`（启动页）

找到这两行（约第 23-24 行）：
```java
private static final String URL_HOME    = "http://192.168.xx.xxx:8080/chat";
private static final String URL_OUTDOOR = "http://100.xx.xxx.xx:8080/chat";
```
- `URL_HOME` → 改成你电脑的 **WiFi 局域网 IP**（cmd 执行 `ipconfig`，找 WLAN 适配器的 IPv4）
- `URL_OUTDOOR` → 改成你电脑的 **Tailscale IP**（不用 Tailscale 可以和 HOME 填一样的）

#### ② `WebViewActivity.java`（fallback 地址）

找到这行（约第 230 行）：
```java
targetUrl = "http://192.168.xx.xxx:8080/chat";
```
改成和 `URL_HOME` 一样的局域网 IP。

#### ③ `AionPushService.java`（推送服务 fallback）

找到这行（约第 187 行）：
```java
String saved = prefs.getString("saved_url", "http://192.168.xx.xxx:8080/chat");
```
同样改成你的局域网 IP。

### 4. 怎么查看自己的 IP？

| IP 类型 | 查看方法 |
|---------|---------|
| WiFi 局域网 IP | 电脑 cmd 执行 `ipconfig`，找到 WLAN 适配器的 IPv4 地址（`192.168.x.x`） |
| Tailscale IP | 安装 Tailscale 后，托盘图标悬浮查看，或访问 https://login.tailscale.com/admin/machines |

> 💡 **什么是 Tailscale？** 一个免费的虚拟组网工具，让你出门在外也能通过手机流量连回家里的电脑。
> 详见 `局域网多端同步方案.md` 第九章。电脑和手机都装上，用同一个账号登录即可。
> 下载地址：https://tailscale.com/download

### 5. 编译安装

修改完 IP 后，在 Android Studio 点菜单 `Build → Build APK(s)`，生成的 APK 在 `app/build/outputs/apk/debug/` 下，传到手机安装即可。

> 手机需要开启「允许安装未知来源应用」，一般在设置 → 安全 → 安装未知应用。

### 6. 常见构建报错

#### 报错：`Dependency requires compileSdk 34` 或 `AAPT: error: resource not found`

这是因为缺少 **Android SDK API 34**。解决方法：
1. Android Studio 菜单 `Tools → SDK Manager`
2. 在 SDK Platforms 标签页，勾选 **Android 14.0 (API 34)**
3. 点 `Apply` 下载安装
4. 重新 Sync Gradle

#### 报错：`Installed Build Tools revision X is corrupted`

1. `Tools → SDK Manager → SDK Tools` 标签页
2. 勾选 **Android SDK Build-Tools 34**
3. Apply 安装后重试

#### 报错：`FAILURE: Build failed - Could not determine java version`

说明 Gradle 用了错误的 JDK 版本。解决方法：
1. `File → Settings → Build, Execution, Deployment → Build Tools → Gradle`
2. 把 `Gradle JDK` 改为 **jbr-17 (Bundled)** 或 **17**
3. 点 Apply 后重新 Sync

#### 报错：`Manifest merger failed` 或 `uses-sdk:minSdkVersion X cannot be smaller than Y`

某个依赖库要求的最低 SDK 版本比项目高。解决方法：
1. 打开 `AionApp/app/build.gradle`
2. 找到 `minSdk 24`，把数字改成报错提示的最小值（比如 `26` 或 `28`）
3. 重新 Sync Gradle

> 提高 `minSdk` 意味着放弃支持更老的手机。`minSdk 24` = Android 7.0+，`minSdk 26` = Android 8.0+。现在大部分手机都是 Android 10 以上，改高一点问题不大。

#### 报错：`AGP version X requires Gradle version Y`

项目使用的 Android Gradle Plugin (AGP 8.2.2) 需要 Gradle 8.2+。如果 Android Studio 提示版本不匹配，点弹窗里的 **"Fix Gradle wrapper"** 按钮自动修复即可。

### 7. 不装 App，只用手机浏览器行不行？

**完全可以！** 手机浏览器访问 `http://电脑IP:8080` 就能聊天，大部分功能都正常。但以下功能**必须依赖 Android App** 才能使用：

| 功能 | 浏览器能用吗？ | 原因 |
|------|---------------|------|
| AI 聊天 / 记忆 / 世界书 / 日程 / 心语日记 | ✅ 完全正常 | 纯 Web 功能 |
| TTS 语音播报 | ✅ 完全正常 | 纯 Web 功能 |
| 音乐点歌 | ⚠️ 仅前台时 | 息屏后浏览器会停止播放 |
| 手机 GPS 持续定位 | ❌ 不可用 | 浏览器后台无法获取 GPS |
| 手机 App 使用记录 | ❌ 不可用 | 需要 Android 系统权限 |
| 锁屏/亮屏检测 | ❌ 不可用 | 需要 Android 系统广播 |
| 闹钟级推送通知（全屏唤醒+震动） | ❌ 不可用 | 浏览器通知无法唤醒屏幕 |
| WebSocket 后台保活 | ❌ 不可用 | 息屏后浏览器会断开连接 |
| 语音唤醒 / 麦克风录音 | ⚠️ 需要 HTTPS | 局域网 HTTP 下浏览器无法调用麦克风 |
| BLE 玩具控制 | ⚠️ 仅 Chrome + HTTPS | Web Bluetooth 兼容性差 |

**简单总结：**
- 💬 **只聊天** → 浏览器完全够用
- 📍 **想让 AI 知道你在哪里、在用什么 App** → 需要装 Android App
- 🔔 **想收到 AI 的闹钟/提醒通知** → 需要装 Android App
